## 1.9 Linking A Javascript File
Adding JS to the HEAD 

```html
<script>
      let js = 'amazing';
      if (js === 'amazing') alert("Javascript is fun!");

      console.log(40 + 8 + 23 - 10);
</script>
```

Adding an external JS File 

```html
<script src="script.js"></script>
```
